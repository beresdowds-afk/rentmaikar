# Hologram Integration

Hologram is the platform's **IoT / eSIM connectivity** provider only.

Hologram MUST NOT participate in:

- User identity verification
- KYC / KYB
- Persona inquiries or templates

Hologram provides:

- eSIM marketplace and inventory
- SIM activation, suspension, plan changes
- Device connectivity (GPS trackers, OBD-II, EV BMS, smart locks, sensors)
- Vehicle-device assignment
- Connectivity dashboards, usage, alerts

See `docs/architecture/HologramResponsibilities.md`
and `docs/architecture/architectureRule.md`.
